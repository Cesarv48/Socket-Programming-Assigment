# Socket Programming Project
**Cesar Villarruel-Reyes**

Implementation of UDP and TCP sockets for client and server sides.

> **Note:** `input.jpeg` must be in the same folder as the client files before running.

## Running the Project

### TCP
```bash
python TCP_server.py
python TCP_client.py
```

### UDP
```bash
python UDP_server.py
python UDP_client.py
```

> **Note:** `input.jpeg` Client side must be ran within 10 seconds of starting server to avoid timeout.
